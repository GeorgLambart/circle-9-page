3. Provision of the Website and Creation of Server Log Files

Each time our website is accessed, our hosting provider automatically collects data and information in so-called server log files. The following data is collected:

  • IP address of the requesting device
  • Date and time of access
  • Name and URL of the accessed file
  • Website from which the access originates (referrer URL)
  • Browser used and, if applicable, operating system
  • Status code of the request

Purpose: The processing is carried out to ensure a smooth connection setup, the use of our website, and to evaluate system security and stability.
Legal Basis: Art. 6(1)(f) GDPR (legitimate interest).
Storage Duration: The data is stored for a maximum of 7 days and then deleted, unless security-relevant events (e.g., attacks) require longer retention.
Recipient: Our hosting provider [e.g., GitHub Inc.], with whom a data processing agreement pursuant to Art. 28 GDPR has been concluded.

5. Your Rights as a Data Subject

Under the GDPR, you have the following rights:

  • Right to Access (Art. 15): You can request information about your processed data.
  • Right to Rectification (Art. 16): You can request correction of inaccurate data.
  • Right to Erasure (Art. 17): You can request the deletion of your data (“right to be forgotten”).
  • Right to Restriction of Processing (Art. 18): You can request the restriction of processing.
  • Right to Data Portability (Art. 20): You can receive your data in a machine-readable format.
  • Right to Object (Art. 21): You can object to processing based on legitimate interest.
  • Right to Lodge a Complaint (Art. 77): You can file a complaint with a supervisory authority, in Austria with the Austrian Data Protection Authority, Barichgasse 40-42, 1030 Vienna, dsb@dsb.gv.at.

To exercise your rights, please contact us at support@circle9games.com.
